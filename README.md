# m-site-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常为静态网页，可能包含信息展示、工具界面或存档内容，不针对任何特定域名或网站。

## 项目简介

m-site-archive 是一个通用的 HTML 页面归档仓库。我们收集、整理并托管一些独立的网页文件，以便于访问和长期保存。这些页面可能是个人项目、临时页面或信息整理页，内容涵盖多种用途，但均以 HTML 文件形式存在，无需后端服务即可直接查看。

## 目录结构说明

```
m-site-archive/
├── pages/           # 存放归档的 HTML 页面文件
├── assets/          # 页面引用的资源文件（如图片、样式、脚本）
└── README.md        # 本文件
```

- `pages/`：每个 HTML 文件或子文件夹代表一个独立的归档页面。
- `assets/`：按需存放页面所需的静态资源，保持文件结构清晰。

## 页面归档说明

归档的 HTML 页面均为静态文件，可直接通过浏览器打开查看。每个页面通常包含完整的 HTML 结构，少数页面可能引用外部资源（如字体、公共库），但尽量保持自包含或使用可访问的 CDN 链接。

页面内容可能来源于个人创作、公开资料整理或临时保存，不涉及商业推广或特定网站关联。归档目的是为了方便访问和备份，不保证内容的长期更新或维护。
